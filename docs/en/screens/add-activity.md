# Add activity

This dialog creates a new activity. You set title, time, reminders and (optionally) a repetition.

## 1) Open
- On **Home**, tap the **+** button.
- Fill the fields and save with **Add**.

## 2) Fields
- **Title** (required), **Description** (optional)
- **Date** and **Time**
- **Repeat** (daily/weekly/custom)
- **Reminders** (one or more)

## 3) Save & done
The activity appears on **Home** and will trigger notifications/alarms as configured.
