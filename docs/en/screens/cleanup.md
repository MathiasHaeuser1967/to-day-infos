# Cleanup

With **Cleanup** you remove activities of a day &ndash; optionally including the associated **notifications/alarms**. The action always applies to the **selected date**.

## 1) Choose date
- **Yesterday / Today / Tomorrow**: quick selection.
- **Date display**: shows the target date to which the cleanup applies.

## 2) Choose what to delete (source)
You choose **exactly one source**. Only entries that **come from this source** on that day will be deleted:

- **Cycle planner** &ndash; activities automatically generated from recurring cycles.  
- **Time announcements** &ndash; activities that were created by time announcements.  

!!! info "Safety"
    Other sources remain untouched. You do not accidentally change content outside the selected source.

## 3) Options

### Only open activities
- **On**: deletes **only** entries that are **not** marked *Done*.  
- **Off**: deletes **all** matching entries &ndash; open **and** done.

### Cancel notifications/alarms
- **On**: ToDay additionally removes **all** notifications belonging to the deleted activities &ndash; both **scheduled** and already **shown**.
- **Off**: Only activities are deleted; existing notifications initially remain visible.

!!! tip
    Enable this if you want to fully clean up "leftover" reminders.

## 4) Execute delete
1. Select date and **source**.  
2. **Check/set** options.  
3. Tap **Delete** &rarr; the dialog closes and you return to **Home**.
