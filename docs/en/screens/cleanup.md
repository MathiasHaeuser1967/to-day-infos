# Cleanup

With **Cleanup** you remove activities of a day – optionally including the associated **notifications/alarms**. The action always applies to the **selected date**.

---

## 1) Choose date

| Option | Description |
|--------|-------------|
| **Yesterday / Today / Tomorrow** | Quick selection. |
| **Date display** | Shows the target date to which the cleanup applies. |

---

## 2) Choose what to delete (source)

You choose **exactly one source**. Only entries that **come from this source** on that day will be deleted:

| Source | Description |
|--------|-------------|
| **Cycle Planner** | Activities automatically generated from recurring cycles. |
| **Time Announcements** | Activities that were created by time announcements. |

!!! info "Safety"
    Other sources remain untouched. You do not accidentally change content outside the selected source.

---

## 3) Options

### Only open activities

| Setting | Effect |
|-------------|---------|
| **On** | Deletes **only** entries that are **not** marked as *Done*. |
| **Off** | Deletes **all** matching entries – open **and** done. |

### Cancel notifications/alarms

| Setting | Effect |
|-------------|---------|
| **On** | Additionally removes **all** notifications belonging to the deleted activities – both **scheduled** and already **shown**. |
| **Off** | Only activities are deleted; existing notifications initially remain visible. |

!!! tip
    Enable this if you want to fully clean up "leftover" reminders.

---

## 4) Execute delete

| Step | Action |
|---------|--------|
| 1 | Select date and **source**. |
| 2 | **Check/set** options. |
| 3 | Tap **Delete** → the dialog closes, and you return to **Home**. |
