# Quick Start

Get up and running in just a few minutes.

---

## 1) Initial Setup

| Step | Action |
|-------|--------|
| 1 | **Start the app** → you land on **Home**. |
| 2 | **Allow notifications** (see documentation System → Permissions) so reminders can be delivered. |
| 3 | Optional: In **Settings → Time Announcements** enable spoken announcements (Text-to-Speech). |

---

## 2) Create Your First Activity

| Step | Action |
|-------|--------|
| 1 | Tap the **Speed Dial** (central button bottom right) and select **➕ New**. |
| 2 | Enter a **Title** (short & clear). |
| 3 | Choose **Date/Time**. |
| 4 | Optionally enable **Repeat** if the activity occurs regularly. |
| 5 | **Save** → the activity appears on **Home**. |

---

## 3) Get Reminded

- Depending on your configuration you receive **notifications/alarms** (status bar) or **time announcements** (spoken hints).  
- Mark activities as **Done** when you are finished.

!!! warning "Background Execution & Power Saving"
    Some manufacturers (e.g., Xiaomi, Huawei, Samsung) restrict apps running in the background.  
    **Solution:** Disable battery optimization for ToDay, allow notifications, and enable *Exact alarms* (if available).
