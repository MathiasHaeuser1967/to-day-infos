# Quick start

Get up and running in a few minutes.

## 1) Initial setup
1. **Start the app** &rarr; you land on **Home**.
2. **Allow notifications** (see documentation System &gt;&gt; Permissions) so reminders can be delivered.
3. Optional: In **Settings &rarr; Time announcements** enable spoken announcements (Text-to-Speech).

## 2) Create your first activity
1. Tap **+ New**.
2. Enter a **Title** (short & unambiguous).
3. Choose **date/time**.
4. Optionally enable **Repeat** if the activity occurs regularly.
5. **Save** &rarr; the activity appears on **Home**.

## 3) Get reminders
- Depending on your configuration you receive **notifications/alarms** (status bar) or **time announcements** (spoken hints).
- Mark activities as **Done** when you are finished.

!!! warning "Background execution & power saving"
    Some manufacturers (e.g., Xiaomi, Huawei, Samsung) restrict apps in the background.  
    **Solution:** Disable battery optimization for ToDay, allow notifications, and enable *Exact alarms* (if available).
